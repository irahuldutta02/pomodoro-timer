## Local Setup

### 1. Clone the repo

```bash
git clone https://github.com/irahuldutta02/pomodoro-timer
cd pomodoro-timer
```

### 2. Install dependencies

```bash
npm install pomodoro-timer
```

### 3. Start the development server

```bash
npm run dev
```
